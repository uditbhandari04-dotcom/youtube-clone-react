# 🎬 YouTube Lite (React Clone)

A simplified YouTube clone built with React using static data. Focused on clean UI, routing, and core features without a backend.

---

## 🚀 Features

* 🏠 Video grid (Home page)
* 🔍 Search (filter from JSON)
* ▶️ Video player (iframe)
* 📂 Sidebar navigation
* 🌙 Dark mode
* 📱 Responsive design

---

## 🛠️ Tech Stack

* React (Vite)
* React Router
* useState, useEffect
* CSS / Tailwind
* Static JSON data

---

## 📁 Structure

```
src/
 ├── components/ (Navbar, Sidebar, VideoCard)
 ├── pages/ (Home, Video)
 ├── data/ (videos.json)
 ├── App.jsx
```

---

## ⚙️ Setup

```
git clone https://github.com/your-username/youtube-lite.git
cd youtube-lite
npm install
npm run dev
```
